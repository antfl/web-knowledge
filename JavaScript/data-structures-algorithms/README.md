# JavaScript 数据结构与算法

数据结构是组织和存储数据的方式，算法是解决问题的步骤。掌握数据结构与算法是成为高级前端工程师的必备技能。

## 数据结构

### 数组 (Array)

数组是最基本的数据结构，用于存储相同类型元素的集合。

**特点：**
- 连续内存存储
- 随机访问时间复杂度 O(1)
- 插入和删除时间复杂度 O(n)

**JavaScript 数组特性：**
- 动态大小
- 可以存储不同类型元素
- 内置丰富的方法

### 链表 (Linked List)

链表由节点组成，每个节点包含数据和指向下一个节点的指针。

**类型：**
- 单向链表
- 双向链表
- 循环链表

**特点：**
- 非连续内存存储
- 插入和删除时间复杂度 O(1)（已知节点）
- 访问时间复杂度 O(n)

### 栈 (Stack)

栈是一种后进先出（LIFO）的数据结构。

**操作：**
- push：入栈
- pop：出栈
- peek：查看栈顶
- isEmpty：判断是否为空

### 队列 (Queue)

队列是一种先进先出（FIFO）的数据结构。

**类型：**
- 普通队列
- 双端队列
- 优先队列

**操作：**
- enqueue：入队
- dequeue：出队
- front：查看队首
- isEmpty：判断是否为空

### 哈希表 (Hash Table)

哈希表通过哈希函数将键映射到值，实现快速查找。

**特点：**
- 平均查找时间复杂度 O(1)
- 处理冲突的方法：链地址法、开放地址法

### 树 (Tree)

树是一种层次结构的数据结构。

**类型：**
- 二叉树
- 二叉搜索树（BST）
- AVL 树
- 红黑树
- 堆
- 字典树（Trie）

**遍历方式：**
- 前序遍历
- 中序遍历
- 后序遍历
- 层序遍历

### 图 (Graph)

图由顶点和边组成，用于表示对象之间的关系。

**表示方式：**
- 邻接矩阵
- 邻接表

**遍历方式：**
- 深度优先搜索（DFS）
- 广度优先搜索（BFS）

## 算法

### 排序算法

**常见排序算法：**
- 冒泡排序
- 选择排序
- 插入排序
- 快速排序
- 归并排序
- 堆排序
- 计数排序
- 桶排序

**时间复杂度：**
| 算法 | 最好 | 平均 | 最坏 | 空间 | 稳定性 |
|------|------|------|------|------|--------|
| 冒泡 | O(n) | O(n²) | O(n²) | O(1) | 稳定 |
| 选择 | O(n²) | O(n²) | O(n²) | O(1) | 不稳定 |
| 插入 | O(n) | O(n²) | O(n²) | O(1) | 稳定 |
| 快速 | O(n log n) | O(n log n) | O(n²) | O(log n) | 不稳定 |
| 归并 | O(n log n) | O(n log n) | O(n log n) | O(n) | 稳定 |
| 堆 | O(n log n) | O(n log n) | O(n log n) | O(1) | 不稳定 |

### 搜索算法

**常见搜索算法：**
- 线性搜索
- 二分搜索
- 哈希搜索
- 树搜索
- 图搜索

### 动态规划 (Dynamic Programming)

动态规划是将问题分解为子问题并存储子问题解的算法技术。

**核心思想：**
- 最优子结构
- 重叠子问题
- 状态转移方程

**常见问题：**
- 斐波那契数列
- 背包问题
- 最长公共子序列
- 爬楼梯问题

### 贪心算法 (Greedy)

贪心算法在每一步选择局部最优解，希望得到全局最优解。

**常见问题：**
- 零钱兑换
- 活动选择
- 分数背包

### 回溯算法 (Backtracking)

回溯算法通过尝试所有可能的解，直到找到符合条件的解。

**常见问题：**
- N 皇后问题
- 子集
- 全排列
- 组合

## 时间复杂度

### 常见时间复杂度

- O(1) - 常数时间
- O(log n) - 对数时间
- O(n) - 线性时间
- O(n log n) - 线性对数时间
- O(n²) - 平方时间
- O(2ⁿ) - 指数时间
- O(n!) - 阶乘时间

## 空间复杂度

空间复杂度衡量算法运行所需的额外空间。
